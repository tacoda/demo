# Demo

_Demo_ is a CLI tool for demonstration purposes to learn about the packagist system.

## Installation

```shell
composer require tacoda/demo
```

## Development

**Install dependencies:**

```shell
make install
```

**Run tests:**

```shell
make test
```

**Run via `make`:**

```shell
make run
```

**Run directly to pass CLI arguments:**

```shell
./bin/demo play
```
